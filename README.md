#H1 Micro(Rest)service seed project

A Restify template seed project, using Typescript.

What you get:
* REST Services: restify
* Language: Typescript (Typings included)
* Client side tooling: gulp
* Logging: bunyan
* Test runner: mocha
* Asserts: chai
* Mocking: sinon

Getting started:

```
npm run restore
npm run build
npm test
npm start
```
